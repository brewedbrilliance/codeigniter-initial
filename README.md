# CodeIgniter4 bootstrap dockerized 

### Requirements:
* docker engine
* docker-compose

## How to run
From your terminal
```
docker-compose build
docker-compse up -d
```

CodeIgniter4 application accessible at http://localhost:8080/public/index.php
